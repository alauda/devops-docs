### New and Optimized Features

#### Alauda DevOps Pipelines

- This update upgrades the catalog versions of existing `Tasks` and `Pipeline` templates, and introduces several new `Tasks`. In the updated versions, default tool images are pinned to fixed tags and some deprecated parameters are removed. Legacy versions are still available, but users are strongly recommended to migrate to the new versions as early as possible. For detailed behavior changes, see the README of each Task and Pipeline template.
