# 功能概述

## 流水线

- Tekton 流水线

  Tekton 的基础，提供自定义资源用于将 CI/CD 工作流定义为一系列步骤、任务和流水线。

- Tekton 触发器

  基于事件的系统，监听外部事件并自动启动流水线执行以响应事件。

- Tekton 链

  供应链安全框架，自动捕获和签署所有流水线执行的来源信息。

- Tekton Hub 集成

  访问可重用任务和流水线的目录，以加速开发。

- Tekton 结果

  对流水线执行历史和日志进行长期存储和查询。

- Webhook 集成

  从 Git 仓库和容器注册表自动触发流水线。

## 工具集成

- 连接器

  提供统一接口，安全地连接到外部工具和平台。从标准协议如 `git` 和 `oci`，平台专注的连接器，到自定义连接器，整个工具链可以轻松地集成到 DevOps 工作流中。

- 连接器类

  创建标准接口，以便在不改变现有连接器的情况下定制和扩展连接器的功能和支持的工具。

- 连接器 CSI 驱动程序

  在云原生工作负载中以无干扰的方式使用连接器，使用您已经熟悉的工具，仅需改变工作负载的配置方式。

## 工具部署与管理

- 模板化部署

  提供部署模板，以便轻松部署包括 Gitlab、Harbor、SonarQube 和 Nexus 等工具。

- 自定义 YAML 部署

  允许用户根据 Helm 图表的值文件自定义工具部署的所有参数。
