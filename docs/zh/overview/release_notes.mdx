# 发版说明

## 4.0.0（Alauda DevOps next-gen）

### 新增及优化功能

#### Alauda DevOps Pipelines v4.0

Alauda DevOps Pipelines 基于 Tekton 及其 API，提供了一种更友好和高效的构建与管理流水线的方式。通过直观的界面和丰富的功能，用户可以轻松创建和管理流水线，并获得实时反馈和洞察。

#### Alauda DevOps Connectors v1.0（Alpha）

连接器提供了一种安全高效的方式连接外部服务和工具，并以不干扰的方式与其集成。平台团队可以轻松安全地创建连接器，同时保护访问秘钥。

#### DevOps 工具链

提供易于部署的 DevOps 工具链实例，并具备一套部署模板，以增加部署速度和信心，同时保持完全的可定制性。包括：

- Alauda Build of Gitlab v17.8
- Alauda Build of Harbor v2.12
- Alauda Build of SonarQube v2025.1
- Alauda Build of Nexus v3.79

提供了从之前的 DevOps 传统工具链迁移的完整逐步指南，具体内容见附加文档。

#### 流水线模板参数

用户可以在流水线模板设计中指定参数值，以便在执行流水线时提供，而无需在基于模板创建流水线时创建和分配参数。此更改使平台构建者能够创建更加灵活和可重用的流水线模板，提高了流水线创建的整体速度。

## 3.20.0（Alauda DevOps）

### 新增及优化功能

#### 工具链实例管理、平台范围集成及管道模板和配置移至 Alauda DevOps

之前的工具链实例管理、平台范围集成以及管道模板和配置属于平台管理模块。随着新版本的发布，该功能已移至 Alauda DevOps。平台管理员现在可以在 Alauda DevOps 内切换到平台管理功能。

#### 项目范围集成及流水线模板移至 Alauda DevOps

之前的项目范围集成和流水线模板属于项目管理模块。随着新版本的发布，该功能已移至 Alauda DevOps。项目管理员现在可以在 Alauda DevOps 内切换到项目管理功能。

#### 流水线模板参数

用户可以在保存流水线模板时指定执行流水线时使用的参数。这一更改使平台构建者能够创建更加灵活和可重用的流水线，提高了流水线创建的整体速度。
