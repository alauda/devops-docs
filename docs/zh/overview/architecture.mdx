# 架构

![](/architecture.drawio.png)

灵雀云 DevOps 由多个插件组成，构建了一个完整的 DevOps 平台，包含以下组件：

- <ExternalSiteLink name="alauda-devops-pipelines" children="灵雀云 DevOps Pipelines" />：基于 Tekton 的云原生管道和任务管理
- <ExternalSiteLink name="alauda-devops-connectors" children="灵雀云 DevOps Connectors" />：安全高效地连接外部服务和 DevOps 工具链
- <ExternalSiteLink name="alauda-build-of-gitlab" children="灵雀云 GitLab 构建" />：开源的基于 Web 的 DevOps 平台，为软件开发团队提供源代码管理、持续集成和问题跟踪功能。
- <ExternalSiteLink name="alauda-build-of-harbor" children="灵雀云 Harbor 构建" />：云原生注册表，安全存储、签署和扫描容器镜像，并具有基于角色的访问控制、漏洞扫描和内容信任能力。
- <ExternalSiteLink name="alauda-build-of-sonarqube" children="灵雀云 SonarQube 构建" />：先进的代码质量和安全平台，集成了 AI 驱动的能力，用于检测和验证 AI 生成的代码，提供全面的漏洞检测以增强安全性，并通过更快的分析时间和无缝的 IDE 集成提高开发者生产力。
- <ExternalSiteLink name="alauda-build-of-nexus" children="灵雀云 Nexus 构建" />：全面的二进制仓库管理器，作为所有内部和第三方二进制文件、组件和包的单一真实来源。
