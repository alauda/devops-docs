# Introduction

## Alauda DevOps Introduction

Alauda DevOps is a container-based cloud application platform that supports unified scheduling of container workloads and fine-grained multi-tenant permissions. It uses the native Kubernetes architecture to orchestrate the DevOps toolchain, which can flexibly and compatibly adapt to customers' existing tool selections, as well as potential future changes in tool selection.
