# Обновление Alauda DevOps

Компоненты Alauda DevOps разработаны так, чтобы автоматически обновляться во время обновлений платформы, не требуя ручного вмешательства. Система обрабатывает все необходимые миграции и обновления без проблем в фоновом режиме.
