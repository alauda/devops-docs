# Обзор возможностей

## Пайплайны

- Tekton Pipelines

  Основу Tekton, которая предоставляет пользовательские ресурсы для определения рабочих процессов CI/CD в виде серии шагов, задач и пайплайнов.

- Tekton Triggers

  Система, основанная на событиях, которая отслеживает внешние события и автоматически запускает выполнение пайплайнов в ответ на них.

- Tekton Chains

  Рамочная структура безопасности поставок, которая автоматически захватывает и подписывает информацию о происхождении для всех выполнений пайплайнов.

- Интеграция Tekton Hub

  Доступ к каталогу повторно используемых задач и пайплайнов для ускорения разработки.

- Tekton Results

  Долговременное хранение и запрос истории выполнения пайплайнов и журналов.

- Интеграция вебхуков

  Автоматический запуск пайплайнов из репозиториев Git и реестров контейнеров.

## Интеграция инструментов

- Коннекторы

  Предоставляет единый интерфейс для безопасного подключения к внешним инструментам и платформам. От стандартных протоколов, таких как `git` и `oci`, до коннекторов, ориентированных на платформу, и пользовательских коннекторов – весь инструментарий можно легко интегрировать в рабочий процесс DevOps.

- Класс коннектора

  Создает стандартный интерфейс для настройки и расширения возможностей коннекторов и поддерживаемых инструментов без изменения существующих коннекторов.

- Драйвер коннектора CSI

  Ненавязчивый способ использования коннекторов в облачных нативных рабочих нагрузках, используя инструменты, которые вы уже знаете, изменяя лишь то, как настраивается рабочая нагрузка.

## Развертывание и управление инструментами

- Развертывание по шаблону

  Предоставляет шаблоны развертывания для легкого развертывания инструментов, включая Gitlab, Harbor, SonarQube и Nexus.

- Пользовательское развертывание YAML

  Позволяет пользователям настраивать все параметры развертывания инструмента на основе файла значений графиков Helm.
